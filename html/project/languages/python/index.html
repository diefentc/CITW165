Secret Santa

During the holidays, our family wanted to do a secret santa gift exchange and we couldn't find a time to get everyone together to draw names. So, I created this program that takes a list of names and emails to assign each person a name to get a gift for. The program then sends an email to each person with the name of their selection. The program allows for a person to not be assigned specific people, helping prevent couples from getting each other gifts.
